# 📊 Dashboard de Vendas em Python

Dashboard interativo para **análise de vendas**, desenvolvido em Python.
O sistema permite visualizar métricas de faturamento, produtos vendidos, regiões e desempenho de vendedores através de gráficos interativos.

Este projeto simula um **sistema de Business Intelligence (BI)** utilizado por empresas para monitorar indicadores de vendas.

---

# 🚀 Funcionalidades

* 📊 Dashboard interativo
* 💰 Indicadores de faturamento
* 📦 Total de produtos vendidos
* 👨‍💼 Análise por vendedor
* 🌎 Análise por região
* 📈 Gráficos interativos
* 🔎 Filtros dinâmicos

---

# 📊 Fonte de dados

O projeto utiliza um conjunto de **dados simulados de vendas**, contendo:

* Data da venda
* Produto
* Categoria
* Quantidade
* Preço
* Vendedor
* Região

Esses dados são utilizados para gerar análises e visualizações no dashboard.

# ⚙️ Como executar o projeto

### 1️⃣ Clonar o repositório

```
git clone https://github.com/seu-usuario/dashboard-vendas-python.git
```

### 2️⃣ Entrar na pasta do projeto

```
cd dashboard-vendas-python
```

### 3️⃣ Instalar as dependências

```
pip install -r requirements.txt
```

### 4️⃣ Executar o dashboard

```
streamlit run dashboard.py
```

O sistema abrirá automaticamente no navegador:

```
http://localhost:8501
```
